Web Security

Web security attacks are very common now days. There are various ways we as developers can prevent this from happening. Sanitazion of data is one of them, it involves destroying or deleting data from storage device, so it cannot be recovered. There are different types of tools that can help you as a developer to prevent attacks. To secure authorization and authentication you can use OneLogin, OneLogin is kind of like a second authentication thing, to secure that its really you that is logging in.
